# DSA Learning Repository

This repository contains C++ practice code for data structures, algorithms, STL examples, and problem solutions.

## Folders

- `BASIC/`: beginner C++ programs and core syntax exercises.
- `CPP STL/`: examples of standard template library containers and operations.
- `PATTERN/`: console pattern-printing exercises.
- `PROBLEMS/`: lecture-based problem solutions and practice code.
- `SORTING/`: sorting algorithm implementations.
- `DSA/`: topic-wise problem sets and solutions.
- `Leetcode problems/`: Leetcode practice files.

## Cleanup Rule

Generated `.exe` files and duplicate `- Copy` files are ignored so the repository stays focused on source code.
